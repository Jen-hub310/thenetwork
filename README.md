### Let's talk about LAGG
#### Not to be confused with the kind of Lag you see when playing video games. This is a different thing. LAGG is 
Link Aggregation. Does it increase bandwidth? Yes, but it's not as straightfoward as some would believe.
It is a per connection increase. Think of each interface and  cable connection as a single lane. 
If you have 2 connections that are 1G each, you have 2G. You can double that with 2 more connections. Why is LaGG good? Increased thruput. Reduncancy. Yay. 
(including me). __Throughput:__ The rate data moves thru a system. An alternative to LAGG? Higher speed interfaces. Think about it. You can have let's say, an
8GB LAGG using 2G interfaces. Or, one 10G interfaced. Easier to go with the one interface at 10G. You should only be using LAGGS if you 
1. Understand the WHY
2. Need them.

Random: port and interface are sometimes ussed interchangeabley.
